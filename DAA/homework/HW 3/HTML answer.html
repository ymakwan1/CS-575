a)

consider A="dog" and B="cof"

consider last characters g and f.

->g can be deleted and we can find edit distance for "do" and "cof".

->g can be replaced by f and we can find edit distance for "do" and "co".

->f can be inserted after g and we can find edit distance for "dog" and "co".

This is optimal substructure

c)

Let string A length be m and B length be n and 

d(i,j) = j if i=0

d(i,j) = i if j=0

d(i,j) = d(i-1,j-1) if i!=0,j!=0 and A[i]=B[j]

d(i,j) = 1+min(d(i-1,j),d(i,j-1),d(i-1,j-1)) otherwise

d)

edit(A[1..m],B[1..n])

create dp[m+1][n+1]

for i=0 to m do

for j=0 to n do

if i=0, then dp[i][j]=j

else if j=0, then dp[i][j] = i

else if A[i]=B[j] then dp[i][j]=dp[i-1][j-1]

else d[i][j] = 1+min(d[i-1][j],d[i][j-1],d[i-1][j-1])

end // j loop

end //i loop

return dp[m][n]
